Module I - Differential amplifier: emitter coupled circuit, common mode and differential mode gain, common mode rejection, complementary output stage, improved differential amplifier with constant current source
Integrated circuits, semiconductor processes, monolithic IC's

Module II - Operational amplifiers: characteristics of ideal op-amps, input, output impedance, offset voltages, frequency compensation, stabilization of gain by negative feedback, inverting and non-inverting amplifiers, current to voltage converter and other configurations

Module III - Applications of Op-amps: analog circuits, unity gain buffer, adding circuits- comparator, integrating and differentiating circuits, logarithmic amplifier, voltage regulator using op-amps, active filters, analogue computer (basic ideas)

Module IV - Digital fundamentals: the binary number system, octal and other codes, binary arithmetic, Boolean algebra, logic fundamentals, Boolean theorems, Boolean functions, synthesis of Boolean functions, logic gates, fundamental logic operations, combinatorial logic circuits, the XOR gate, the Karnaugh diagram

Module V - Logic circuits: the NOT gate and inversion operation, diode-transistor logic, emitter coupled logic gates, CMOS inverter circuits, NAND and NOR CMOS switches, the transmission gate- gate arrays


References:
1. Electronic Fundamental and Applications - J. D. Ryder (Text)
2. Digital Electronics - C. E. Strangio
3. Integrated Electronics - Millman and Halkias (Text)
