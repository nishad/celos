Module I - Photonics Technology: components, couplers, isolators, circulators, multiplexers and filters, fibre gratings, interferometers, FO amplifiers, transmitters and detectors, switches, wavelength converters, nonlinear effects in signal transmission, self phase and cross phase modulation, soliton pulse propagation

Module II - Modulation and demodulation: signal formats, direction detection receivers, coherent detection, test beds, LAMBDANET, STARNET, rainbow, wavelength routing network, optical layer in network, node design, Networking design and operation, routing wavelength assignment

Module III - Control and management: network management function, configuration, performance, and fault managements, optical safety
Wavelength routing test beds: AON, NTTR, ONTC, MONET

Module IV - Access network: network architecture, HFC, FTTC, optical access network architecture, deployment considerations, upgrading the transmission capacity, SDM, TDM, WDM, application areas, inter exchange, undersea, local exchange networks

Module V - Packaging and cabling of photonics components: photonic packet switching, OTDM, multiplexing and demultiplexing, optical logic gates, synchronization, broadcast OTDM network, OTDM testbeds


References:
1. Optical Networks: A Practical applications - R Ramaswami and K N Sivarajan
2. Photonic Switching Technology - H T Mouftah, J M H Elmirghani
3. Deploying Optical Networking Components - Gil Held
4. Optical Interconnection - C Tocci, H J Caulfield
