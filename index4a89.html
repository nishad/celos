Module I - Computers and peripherals: processor, CPU, ALU, registers, storage, software, operating systems, application software, supercomputers, mainframe computers, mini computers, workstations, desktop models, notebook computers, network computers, handheld PCs, standard, ergonomic and specialty keyboards, mouse, trackball, trackpad, trackpointer, pens, touch screens, game controllers, bar code readers, image scanners, OCR, microphones, speech recognition, video input, digital cameras, CRT and flat panel monitors, paper-white displays, ELD displays, plasma/gas plasma displays, comparing monitors, video controller, sound systems, DMP, inkjet, laser, snapshot, thermal-wax, dye-sub, Fiery & IRIS printers, plotters, PC-to-TV converters

Module II - Programming: algorithm concepts, creation of simple programs in VB.net, system development life cycle

Module III - Processors and storage: memory, ROM, SRAM, DRAM, flash memory, how registers affect speed?, memory and computing power, internal clock, Bus and Bus types, cache memory, USB, serial, parallel, SCSI and MIDI ports, PCMCIA cards, expansion slots and boards, plug and play, MIDI devices, modern processors and manufactures, RISC processors, parallel processing, future of processing, magnetic storage devices, magnetic disks, data organization, how OS finds data on a disk?, diskettes, their types and capacities, hard disks, removable high capacity disks, tape drives, DAT, medium, software and schedule for backup, optical storage devices, CD-ROM, DVD-ROM, CD-recordable, CD-rewritable, photo CD, average access time, file compression, data transfer rate, EIDE, SCSI - the future of data storage

Module IV - Operating systems: user interfaces, command line interfaces, OS services, sharing information, multitasking, drivers, utilities for file defragmentation, data compression, backup, antivirus and screen savers, brief introduction to Windows-XP, -NT, -NT workstation, -NT server, -95, -98, -2000, Unix, Linux, Mac, OS/2 Warp, word processing, spread sheet, and DTP s/w, adding graphics and sounds, converting documents into web pages, presentation programs, e-books, DBMS, file formats like .doc, .xls, .mdb, .gif, .jpeg (.jpg), .png, .bmp, .tiff (.tif), .pcx, .ppm, .tga, .ai, .cmx, .eps, .wpg, .cgm .mpg, .wav, .au .snd, .ra, .mid .avi, .qt, .aif, .mov, .rm, .swf

Module V - Internet and other computing issues: LANs, WANs, client/server and peer-to-peer networks, network topologies, cables, links, cards and network software, modem, ISDN, DSL, ATM, cable modem, how internet works?, TCP/IP, traffic, IP and DNS addresses, domains and sub domains, www, browsers, HTML tags, HTTP, URLs, home page streaming technology, search engines, e-mail, listserv, newsgroups, telnet, FTP, IRC, online services, direct, remote and gateway connections, internet through LAN, firewalls, intranets, extranets, overview of e-commerce, security, ergonomics and health issues, privacy, piracy and virus issues, theft, ethical issues in computing

Advanced Reading section - FORTRAN, C++, MATLAB


Lab work:
1. At least 20 basic VB.net programs, mostly on Photonics and related topics
2. Simple exercises on Windows-XP, Microsoft Word-2003, Excel-2003, Access-2003, Powerpoint-2003
3. Exercises on the basic commands of UNIX & Linux


References:

3. Computers Today - Suresh K. Basandra
4. How Computers Work - Ron White
5. VB.net programming black book - Steven Holzner
