Module I - Arithmetic logic units, half adders and full adders, 2’s complement, 2’s complement adder-subtractor, flip-flop, R-S latches, level clocking, D latch, edge triggered D flip flops, J-K flip-flop, J-K master slave flip-flop

Module II - Registers and counters: buffer register, shift register, controlled shift registers, ripple counters, synchronous counters, ring counters, three state registers, Bus organized computers, architecture of simple computers, instruction set, programming steps, fetch cycle, execution cycle, schematic diagram, micro-programming

Module III - Programming microprocessors: bi directional registers, memory reference instructions, register instructions, jump call instructions, logic instructions, programming model, arithmetic instructions, increments, decrements, rotates, logic instructions, extended register memory, stack instructions

Module IV - Microprocessors 8085: block diagram, pin out diagram, minimum system, fetching and executing, timing diagrams, I/O operation, programmed I/O, interrupts, serial input and serial output, extending the interrupt system, direct memory access

Module V - Peripheral operations: programming I/O ports, time expanding memory, static RAMs, dynamic RAMs, basic D/A converter, A/D converters, counter method, successive approximation, VCO, simple and hold circuits


References:
1. Microprocessor Architecture: Programming and Applications using 8085 - R S Goanker
2. Digital Computer Electronics - Malvino (Text)
3. Integrated Electronics - Milman and Halkias
4. Digital Principles and Applications - Malvino and Leach, McGrowhill (Text)
5. Electronic Devices: Applications and Integrated Circuits - Mathur Kulashreshta and Chadha
